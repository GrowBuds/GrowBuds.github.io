About

LetUmGrow is a social network for plants located on the UH Manoa Campus.

Looking for a specific plant or want to learn more about a plant that you've seen? Our interactive map displays the location of nearby plants and links to each plant profile.

Spotted a new plant on campus? Create a profile page for that plant and add updates to its feed, like pictures or statuses. Who knows, there might even be a reward waiting for you if you do!

Guided Tour

Landing

This is the initial page a user will see when visiting the LetUmGrow site. This page is used to log in.

Home

This is the page users will be directed to after logging in and agreeing to terms of service.

Tutorial

The tutorial page displays the process of how to use the LetUmGrow app.

Add Plant

The Add Plants page shows your current location on the map displayed, and also allows you to add a plant to the database given it's latitude and longitude coordinates. The plant's scientific name, vernacular name, and description are optional.

Plant Map

The Plant Map page allows the user to plot locations of specific plants by clicking on the map and dropping pins. The pins can also be moved by clicking on them and dragging them. These pins are saved to the database and displayed on the List Plants page.

List Plants

The List Plant page displays the pins' locations (longitude and latitude) that were dropped on the Plant Map page, or added via the form on the Add Plants page. You can also edit the information of any plant listed on the page.

If you click on the 'Profile' button in the bottom left-hand corner of a plant listed, you will be taken to that specific plant's profile

Plant Profile

The Plant Profile contains specific information about a certain plant and it's location on the map.

User Profile

After logging in the user is guided to the User Profile page and the top menu becomes available, which the following content will explain in further detail. Note: the 'Logged In' button contains a drop-down item which allows you to logout and return back to the landing page.

Remedies

The Remedy page is a reference to Lāʻau Lapaʻau (Hawaiian Medicine) remedies, materials needed, and procedures of preparation and application. Techniques used in Lāʻau Lapaʻau are also included as well. Click on dropdown items to view the content.

FAQ

The FAQ page contains frequently asked questions separated in sections site, accounts, and mobile. Simply click on the item you prefer to view and it's contents will show in a drop-down.
